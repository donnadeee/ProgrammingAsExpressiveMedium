Computer art

  • Computers have only been around since the late 30's.
  • Computer art is a new form of art.
  • Music, dance, poetry, sculpture, and drama are art forms that have been around for thousands of years. New forms of art are rare.
  • What makes computer art a new form of art? Have a read of Computer Art and the Theory of Computation.

What is JavaScript?

  • You will be studying JavaScript programming.
  • JavaScript is the programming language all web browsers 'understand'.
  • JavaScript is a totally different programming language from Java.
  • HTML and CSS are not programming languages.
  • HTML and CSS are primarily static.
  • JavaScript (and other programming languages) provide dynamism, the ability to change things on the fly, in response to user input or other input such as network conditions.

The canvas tag

  • The <canvas> tag is an HTML 5 addition.
  • It has been touted as a replacement for Flash.
  • It isn't as featureful as Flash (yet) but it's a good start.
  • I created Aleph Null using the canvas tag.
  • Check out http://html5canvastutorials.com/tutorials/html5-canvas-element. This is a great interactive tutorial about the canvas tag.

20 steps to your first JavaScript program

  1. Yer basic HTML file
  2. Put in the canvas tag
  3. Put in the style sheet
  4. Put in the script tag
  5. Get a handle on the canvas
  6. Set the lineWidth and create an object
  7. Track mouse movement
  8. Calling a function that draws a line
  9. Remember the old position of the mouse
  10. Oops this is the same as 9!
  11. Let's write a function that draws a line
  12. If there is an old mouse value then...
  13. Begin a path
  14. Make the start point the old mouse position
  15. The path now goes from that point to the current mouse position
  16. Stroke the path
  17. React when the mouse leaves the canvas
  18. Set m.oldX to null
  19. Set m.oldY to null
  20. Change the line color to white
